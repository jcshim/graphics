## 🎯 강의 목표 (개조식 4개)

* 컴퓨터 그래픽스의 기본 개념과 3D 공간 구조를 이해하고 시각적으로 표현할 수 있도록 한다.
* 웹 기반 그래픽스 도구를 활용하여 이론과 실습을 연계한 그래픽스 콘텐츠를 구현할 수 있도록 한다.
* 사용자 인터랙션과 애니메이션을 포함한 실시간 그래픽스 프로그래밍 능력을 기른다.
* AI·메타버스 등 최신 기술과 그래픽스를 융합하여 창의적인 디지털 콘텐츠를 설계하고 표현할 수 있도록 한다.

## 📘 강의 요약 (개조식 6줄)

* 본 교과목은 컴퓨터공학 전공 학생을 대상으로 이론과 실습을 병행하는 그래픽스 수업이다.
* 그래픽스의 기초 개념을 바탕으로 3D 공간, 카메라, 조명, 애니메이션 등 핵심 요소를 단계적으로 학습한다.
* 웹 기반 그래픽스 환경을 활용하여 별도의 복잡한 설치 없이 실습 중심으로 수업을 진행한다.
* 실시간 렌더링과 사용자 인터랙션을 통해 그래픽스의 동작 원리를 직관적으로 이해하도록 한다.
* AI 기반 시각 인식과 메타버스 개념을 접목하여 그래픽스의 최신 응용 분야를 경험한다.
* 개인 및 팀 프로젝트를 통해 학습 내용을 종합하고 결과물을 포트폴리오로 완성한다.
