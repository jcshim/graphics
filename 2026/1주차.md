## 🟦 Slide 3. 컴퓨터 그래픽스의 정의

> 컴퓨터를 이용하여 2D 및 3D 영상과 이미지를 생성·처리·표현하는 기술 및 학문 분야

* 수학적 모델 기반
* 알고리즘 중심
* 빛과 공간 계산

간단 정의:
**“컴퓨터로 그림을 그리고, 빛을 계산하고, 가상 세계를 만드는 학문”**

---

## 🟦 Slide 4. 그래픽스는 무엇을 계산하는가?

* 위치(Position)
* 변환(Transformation)
* 조명(Lighting)
* 색상(Color)
* 가시성(Visibility)

👉 핵심: **빛과 공간의 계산**

---

## 🟦 Slide 5. 그래픽스의 기본 구조 (Pipeline)

```
모델링 → 변환 → 조명 → 렌더링 → 화면 출력
```

또는

```
Vertex → Clipping → Rasterization → Fragment → Display
```

👉 이것이 한 학기 동안 배울 핵심 흐름

* 2010s: WebGL, 모바일
